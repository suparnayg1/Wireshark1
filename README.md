# Wireshark1
Network analysis project
Wireshark Network Traffic Analysis Project
## 📌 Project Overview
This project demonstrates basic network traffic analysis using *Wireshark*.  
The goal is to capture, filter, and analyze network packets to understand how data flows across a network.
This project is created as part of my learning journey in *Cyber Security and Networking fundamentals*.
## 🛠️ Tools Used
- *Wireshark* – Packet capturing and analysis tool  
- *Operating System* – Linux / Windows  
- *Network Interface* – Ethernet / Wi-Fi

- Packet capture screenshots
- Filtered traffic analysis
- Graphical representation of network traffic
- README documentation
